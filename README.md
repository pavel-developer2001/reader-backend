# Бэкенд для Reader

## Версия 0.3.0

## В разработке применяется:

- Node js
- Express
- Sequelize, PostgreSQL
- Cloudinary

## Установка

- 1 Надо установить все пакеты:

```bash
npm i
# or
yarn install
```

- 2 Запускаем приложение:

```bash
npm run server
# or
yarn server
```

Фронтенд проекта [здесь](https://github.com/pavel-developer2001/reader-frontend)
